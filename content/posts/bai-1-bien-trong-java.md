Trong lập trình Java, biến là nơi lưu trữ giá trị. Java là một ngôn ngữ có kiểu dữ liệu mạnh (strongly typed), nghĩa là bạn phải khai báo kiểu dữ liệu cho biến.
- **int**: Số nguyên (VD: 1, 100)
- **double**: Số thực (VD: 10.5)
- **String**: Chuỗi văn bản (VD: "Hello Java")
- **boolean**: Giá trị đúng/sai (true/false)
